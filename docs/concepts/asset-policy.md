# Asset Policy

## Regra principal

O slice deve usar assets livres, com preferencia forte por CC0.

## Diretrizes

- Registrar toda fonte em `public/assets/SOURCES.md`.
- Priorizar coesao visual sobre variedade.
- Curar apenas os arquivos usados no runtime final.
- Fallbacks devem ser documentados.

## Alvo atual

- Tileset de mundo top-down coerente com fazenda/vila.
- Sprite de personagem com caminhada em quatro direcoes.

## Escolha atual

- Mundo: `Puny World Tileset`
- Personagem base: `Puny Characters / Character-Base`
- Variante NPC candidata: `Puny Characters / Soldier-Blue`
